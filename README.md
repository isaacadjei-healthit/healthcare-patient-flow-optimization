# Healthcare Patient Flow Optimization System

## Overview
A healthcare database management system designed to optimize patient flow, resource allocation, and operational analytics using SQL Server.

## Technologies Used
- SQL Server
- Azure Data Studio
- Docker
- T-SQL
- Database Normalization (3NF)

## Key Features
- Patient wait time analytics
- Length of stay reporting
- Staff workload analysis
- Resource utilization metrics
- Healthcare dashboard KPIs

## Database Tables
- Patients
- Visits
- Staff
- Resources
- Visit_Staff
- Visit_Resources

## Environment
Built using SQL Server running inside Docker and managed through Azure Data Studio.

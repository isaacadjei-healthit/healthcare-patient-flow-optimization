# Project Overview

This project demonstrates the design and implementation of a healthcare database management system focused on patient flow optimization.

The database was developed using SQL Server running inside Docker and managed using Azure Data Studio.

The system supports:
- Patient admissions tracking
- Staff assignment tracking
- Resource allocation monitoring
- Length of stay analytics
- Wait time analysis
- Healthcare operational dashboard reporting

The project applies:
- Relational database modeling
- Third Normal Form (3NF)
- Primary and foreign key constraints
- SQL analytics queries
- Healthcare operational reporting concepts
